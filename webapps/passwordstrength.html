Enter a password first to see strength.

How the Password Strength Calculation Works

This password strength calculator estimates how long a password would take to crack, assuming the rate of password guesses doubles every two years, following an interpretation of Moore's Law. Furthermore, it considers the cumulative count of every guesses made over time, rather than just the current guess rate.

The valid characters in this password strength checker are the 95 ASCII text characters and the space character: 

abcdefghijklmnopqrstuvwxyz
ABCDEFGHIJKLMNOPQRSTUVWXYZ
1234567890
!"#$%&'()*+,-./:<=>?@[\]^_`{|}~

Security

This web app does NOT send the password to a server for strength calculation. All computation is performed locally.

Limitations

This program has a hard coded starting guessing rate and a hard coded guessing rate doubling time scale. It also ignores password guessing techniques such as dictionary lookup. The output of the program is not guaranteed to be correct. Predicting the future correctly is impossible.
