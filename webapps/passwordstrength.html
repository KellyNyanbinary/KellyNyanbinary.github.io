Password Strength Checker

Enter a password first to see strength.

This is a password strength calculator that assumes the rate of password guesses doubles every other year (Moore's Law). Furthermore, it takes the integral of every guess taken so far. The starting guess rate is 500,000 guesses per second, taken from Phoronix's OpenBenchmarking database.

This webapp does NOT send the password to a server for strength calculation. All computation is performed locally.
