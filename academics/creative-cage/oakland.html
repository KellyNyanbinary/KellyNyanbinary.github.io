Aurora Paint, Perpetual

A warehouse loft full of solder smoke and synth playlists becomes the HQ for a scrappy GPLv3 alternative.

Build Log

  • Version control: Self-hosted GitLab.
  • License: GPLv3.
  • Project codename: Aurora Paint. Motto: “Art should not expire.”

Miguel + Mei

Miguel taps code with his foot keeping tempo; Mei rewrites color pipeline logic without license locks. Donations trickle in, proof that users will fund tools that promise permanence.

They cite Krita’s success as validation.

Homepage Copy
